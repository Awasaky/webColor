# webColor
translate colors from web to openGL format

Usage:

local webColor = require("webColor")
print(webColor("#BAB"))
print(webColor("#BABF00"))
print(webColor("BABF00"))
print(webColor("BABF00DD"))
print(webColor("#BABF00DD"))

Can work with 3, 4, 6, 8 letters colors
Understand staring symbol # too
